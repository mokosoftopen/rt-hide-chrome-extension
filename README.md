# How to Install
- git clone するか、zip をダウンロードして展開
- Chrome の設定画面を開く (Macの場合 [Chome]->[環境設定...])
- 拡張機能を選択
- [デベロッパー モード] をオンにする
- [パッケージ化されていない拡張機能を読み込む...] ボタンを押す
- ダウンロードしたディレクトリを選択
